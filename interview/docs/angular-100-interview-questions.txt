100 Important Angular Interview Questions and Answers

1. What is Angular?
Answer: Angular is a TypeScript-based open-source front-end framework developed by Google used to build single-page applications (SPA).

2. What is TypeScript and why Angular uses it?
Answer: TypeScript is a superset of JavaScript that adds static typing, classes, and interfaces. Angular uses it for better tooling and maintainability.

3. What are Angular components?
Answer: Components are the building blocks of Angular applications. They control views using templates and logic.

4. What is a module in Angular?
Answer: An Angular module (NgModule) organizes components, directives, pipes, and services into cohesive blocks.

5. What is data binding?
Answer: Data binding synchronizes data between component and template.

6. Types of data binding?
Answer: Interpolation, Property Binding, Event Binding, and Two-way Binding.

7. What is interpolation?
Answer: Interpolation binds component data to template using {{ }}.

8. What is property binding?
Answer: Property binding binds component values to DOM properties using [].

9. What is event binding?
Answer: Event binding listens to DOM events using ().

10. What is two-way data binding?
Answer: Two-way binding combines property and event binding using [(ngModel)].

11. What is a directive?
Answer: Directives modify DOM behavior or structure.

12. Types of directives?
Answer: Component directives, Structural directives, Attribute directives.

13. What is ngIf?
Answer: Structural directive used to conditionally add/remove elements from DOM.

14. What is ngFor?
Answer: Structural directive used to loop through collections.

15. What is ngClass?
Answer: Directive used to dynamically add/remove CSS classes.

16. What is ngStyle?
Answer: Directive used to dynamically apply styles.

17. What is dependency injection?
Answer: A design pattern where dependencies are injected into components rather than created inside them.

18. What is a service?
Answer: A reusable class used for business logic and shared data.

Answer: Command Line Interface used to create, build, and manage Angular projects.
