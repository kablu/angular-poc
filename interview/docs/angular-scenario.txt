File: angular-scenario.txt

Scenario: Child to Parent Data Transfer in Angular

Question:

You are developing an e-commerce Angular application.

There are two components:
1. ProductListComponent (Parent)
2. ProductItemComponent (Child)

Each product item has an "Add to Cart" button.

Requirement:
When the user clicks the "Add to Cart" button in the child component,
the product ID should be sent to the parent component and the parent
should update the cart count.

How would you implement this in Angular?


Answer:

Angular supports Child → Parent communication using @Output() and EventEmitter.

Step 1: Child Component

product-item.component.ts

import { Component, Input, Output, EventEmitter } from '@angular/core';

@Component({
  selector: 'app-product-item',
  template: `
    <div>
      <h3>{{productName}}</h3>
      <button (click)="addToCart()">Add to Cart</button>
    </div>
  `
})
export class ProductItemComponent {

  @Input() productId!: number;
  @Input() productName!: string;

  @Output() cartEvent = new EventEmitter<number>();

  addToCart(){
    this.cartEvent.emit(this.productId);
  }

}


Step 2: Parent Component

product-list.component.ts

import { Component } from '@angular/core';

@Component({
  selector: 'app-product-list',
  templateUrl: './product-list.component.html'
})
export class ProductListComponent {

  cartCount = 0;

  addProductToCart(productId:number){
    console.log("Product added:", productId);
    this.cartCount++;
  }

}


Step 3: Parent Template

product-list.component.html

<h2>Cart Items: {{cartCount}}</h2>

<app-product-item
  [productId]="1"
  productName="Laptop"
  (cartEvent)="addProductToCart($event)">
</app-product-item>

<app-product-item
  [productId]="2"
  productName="Mobile"
  (cartEvent)="addProductToCart($event)">
</app-product-item>


Flow Explanation:

Child Component
    |
    | emit(productId)
    ↓
Parent Template Event Binding
    |
    | (cartEvent)="addProductToCart($event)"
    ↓
Parent Component Method Executes


Important Interview Points:

1. @Output() is used to send data from child to parent.
2. EventEmitter emits the event.
3. $event contains the value sent by the child component.
4. Parent listens to the event using event binding.
